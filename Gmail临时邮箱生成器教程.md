## 📚 背景知识：Gmail 的点（.）规则

在深入教程前，需要了解 Gmail 的一个重要特性：

> **Gmail 地址中的点（.）不影响投递**  
> `john.smith@gmail.com` = `johnsmith@gmail.com`

这意味着您可以通过**在用户名中随意添加点**来创建无限别名，所有这些别名都会直达您的主邮箱。这是 Google 官方确认的行为。

**示例**：
- 主邮箱：`myemail@gmail.com`
- 临时别名：
  - `my.email@gmail.com`
  - `my.e.mail@gmail.com`
  - `myem.ail@gmail.com`
  - 等等...

所有发往这些别名的邮件都会出现在 `myemail@gmail.com` 的收件箱中。

## 💡 方法二：手动创建（无需工具）

如果您不想依赖第三方应用，可以手动创建临时别名：

### 步骤
1. 确定您的主 Gmail：`username@gmail.com`
2. 在用户名中添加任意数量的点（.）：
   - `user.name@gmail.com`
   - `u.s.e.r.n.a.m.e@gmail.com`
   - 甚至 `u。。。。s。。。。e。。。。r。。。。`（可以使用 Unicode 点，但某些网站可能拒绝）
3. 在注册网站时使用这个新地址
4. **在 Gmail 中创建过滤器**，自动处理这些邮件：

```gmail-filter
搜索条件: to:user.name@gmail.com
操作: 跳过收件箱, 标记为已读, 应用标签"临时注册"
```

### 注意事项
- 不要忘记在过滤器中包含**所有您创建的变体**，或者使用更通用的模式
- 某些网站可能**拒绝**包含点的邮箱格式（虽然 Gmail 支持，但网站前端可能限制）
- 如果使用 Unicode 点，发送方可能无法正常显示地址

## 🔍 实用技巧与注意事项

### ✅ 推荐做法
1. **为不同用途使用不同模式**：
   - `my.email+news@gmail.com` → 新闻订阅
   - `my.email+forum@gmail.com` → 论坛注册
   - `my.email+shop@gmail.com` → 电商网站
   - 通过在点后加 `+` 前缀可以便于识别

2. **在过滤器中用"或"条件**：
   ```
   to:my.email+news@gmail.com OR to:my.email+forum@gmail.com
   ```
   可以批量管理同一类别的临时地址

3. **保留原始主邮箱用于重要通信**：
   - 银行、工作、正式场合不要使用临时地址
   - 仅用于**潜在垃圾邮件源**的注册

4. **定期检查过滤器**：
   - 确保没有重要邮件被误过滤
   - 根据收到的新类型邮件调整规则

### ⚠️ 风险与限制
| 风险 | 说明 | 规避方法 |
|------|------|----------|
| 网站拒绝点地址 | 某些表单会验证邮箱格式 | 使用标准格式，或换一个点组合 |
| 回复地址显示问题 | 回复可能从临时地址发出 | 在 Gmail 设置中调整"发送邮件为"地址 |
| 忘记过滤器规则 | 导致临时邮件堆积 | 在 Aspose 应用或笔记中记录所有临时地址 |
| 账户安全 | 如果主邮箱被盗，所有临时地址受影响 | 启用 2FA，保持主邮箱安全 |

### 📊 支持的网站类型
- ✅ 社交媒体（Facebook、Twitter、Instagram 等）
- ✅ 论坛和社区（Reddit、Discourse 等）
- ✅ 电商平台（Amazon、淘宝等）
- ✅ 新闻订阅（新闻简报、RSS 推送）

  - 政府网站

## 🎯 进阶用法

### 1. 批量自动化（脚本生成）
如果您需要大量临时地址，可以写脚本自动生成带点的变体：

```bash
# 示例：生成 "john.doe" 的所有点组合
python3 -c "
base='john'
tld='doe'
for i in range(1<<5):
    bits = [(i>>j)&1 for j in range(5)]
    dots = ''.join(['.' if bit else '' for bit in bits])
    print(f'{base}{dots}{tld}@gmail.com')
"
```

### 2. Gmail 过滤器高级语法
利用 Gmail 强大的搜索语法：

```gmail-search
# 匹配所有带点的变体（但不包括无点版本）
to:john.doe@gmail.com OR to:john..doe@gmail.com

# 匹配 + 后缀（推荐更易识别）
to:john.doe+*@gmail.com
```

### 3. 自动回复设置
为临时地址设置自动回复，告知发件人这是临时地址：

在 Gmail → 设置 → 滤镜 → 选择已创建的临时地址过滤器 → 勾选"发送模板回复" → 创建回复模板：

> "此邮箱为临时注册地址，回复可能不会及时处理。如需联系，请使用主邮箱：myreal@gmail.com"

## ❓ 常见问题

### Q1: Gmail 的点规则是 Google 官方支持的吗？
**是的**。Google 官方确认：https://support.google.com/mail/answer/7436150

### Q2: 所有邮件都会到达主邮箱吗？
**是的**。任何带点变体的邮件都会投递到与无点版本相同的收件箱。

### Q3: 其他人能否注册我的点变体地址？
**不能**。如果您拥有 `john.doe@gmail.com`，其他人无法注册 `j.o.h.n.d.o.e@gmail.com`。系统会提示"用户名已被占用"。

### Q4: Aspose 生成器会访问我的 Gmail 吗？
**不会**。它仅生成地址和过滤器说明，不要求您授予其 Gmail 访问权限，因此非常安全。

### Q5: 临时地址可以用于密码重置吗？
**可以，但不推荐**。如果网站向临时地址发送重置链接，您仍能收到。但建议：
- 重要账户绑定主邮箱
- 临时地址仅用于低风险注册

### Q6: 如何处理临时地址的回复邮件？
- Gmail 会自动将发往临时地址的邮件显示在主邮箱
- 回复时，Gmail 默认使用您的主邮箱作为"发件人"
- 如需以临时地址回复，需要在 Gmail 设置中配置"发送邮件为"额外地址
