## 📚 什么是 Discourse？

Discourse 是**开源的现代论坛软件**，被成千上万社区采用。与传统论坛（如 phpBB）不同，Discourse 提供：
- 实时更新的流式讨论
- 强大的通知系统
- 基于信任等级的权限管理
- 优雅的 Markdown 支持
- 移动端原生体验

官方托管服务：https://meta.discourse.org

## 🚀 2 分钟创建您的论坛

### 步骤 1：访问注册页面

打开：https://meta.discourse.org  
点击右上角 **"Sign up"**（注册）

### 步骤 2：使用 Google 账号一键注册

1. 点击 **"Sign up with Google"** 按钮
2. 在弹出的 Google 授权窗口中，选择您的 Google 账号
3. 授权 Discourse 访问基本资料（邮箱、姓名）
4. 自动完成注册，无需填写表单

### 步骤 3：创建您的社区

1. 登录后，点击右上角头像 → **"Create Community"**（创建社区）
2. 填写基本信息：
   - **社区名称**（如"OpenClaw 用户社区"）
   - **子域名**（如 `openclaw.discourse.org` → 完整地址 `https://openclaw.discourse.org`）
   - **描述**：简短说明社区主题
3. 点击 **"Create Site"**（创建站点）

### 步骤 4：初始配置（可选）

系统将自动创建论坛，您可以：
- 上传 logo 和横幅
- 调整主题颜色
- 邀请首批成员（通过邮件邀请）
- 设置分类和标签

**论坛即刻上线！** 🎉 您的专属社区地址：`https://您的子域名.discourse.org`

## ⚙️ 进阶配置

### 自定义域名（付费功能）

如果您有自己的域名（如 `forum.example.com`），可以：
1. 升级到付费计划
2. 在 Discourse 设置中添加自定义域名
3. 配置 DNS CNAME 指向 Discourse
4. 自动获得 SSL 证书

### 邀请与成员管理

- **邀请链接**：设置 → 邀请 → 生成邀请码
- **批量邀请**：上传 CSV 列表，系统自动发送邀请邮件
- **权限控制**：
  - 新成员默认只能阅读
  - 达到一定信任等级后可发帖
  - 管理员可手动提升权限

### 主题与插件

免费计划提供：
- 多种官方主题（Light、Dark 等）
- 常用插件（如投票、摘要、警告）

付费计划可安装第三方插件。

## 💡 使用技巧

### 1. 善用标签和分类
- 创建清晰的分类（如"公告"、"技术讨论"、"反馈建议"）
- 使用标签标记话题类型（如"教程"、"问题"、"想法"）

### 2. 设置信任等级
 Discourse 的信任等级自动提升，您可以调整：
- 新用户：仅阅读，不能发帖
- Level 1：可发帖，但不能上传图片
- Level 2：可发帖+上传图片
- 高级别：可管理内容

### 3. 邮件通知优化
- 鼓励成员设置个人通知偏好
- 重要公告使用"全局公告"功能
- 避免滥用@提及

### 4. SEO 设置
- 在"设置 → 基本"中填写站点标题和描述
- Discourse 自动生成站点地图（/sitemap.xml）
- 启用"公开搜索引擎索引"

## 🔧 常见问题

### Q1: 免费计划会有广告吗？
**没有**。Discourse 托管服务不插入广告，无论免费还是付费。

### Q2: 可以迁移到自托管吗？
**可以**。Discourse 提供完整的导出功能，您可以随时：
1. 在设置中导出所有数据（SQL 转储 + 附件）
2. 自行部署 Discourse（需要服务器技术知识）
3. 导入数据

### Q3: Google 注册后可以改密码吗？
**可以**。在"设置 → 密码"中设置独立密码，之后可以用邮箱+密码登录，无需依赖 Google。

### Q4: 支持其他 OAuth 提供商吗？
**支持**。Discourse 原生支持：
- Google
- GitHub
- Twitter/X
- Facebook
- Apple
- 以及任何符合 OAuth2 标准的提供商

### Q5: 如何备份论坛数据？
免费计划由 Discourse 自动备份（保留 14 天）。付费计划：
- 每日自动备份
- 可手动下载备份文件
- 支持外部 S3 存储

### Q6: 遇到技术问题怎么办？
- 首先访问 [Discourse Meta](https://meta.discourse.org) 搜索解决方案
- 官方文档：https://docs.discourse.org/
- 付费用户享有优先支持
