## 🚀 快速开始

### 1️⃣ 克隆仓库

```bash
# GitHub
git clone https://github.com/gitmeus/openclaw-experience-summary.git

# Gitee（国内访问更快）
git clone https://gitee.com/ztycttnk/openclaw-tutorial-collection.git
```

### 2️⃣ 选择教程

进入仓库目录，打开对应 Markdown 文件即可阅读：

```bash

open "小米音箱接入AI教程.md"
```
